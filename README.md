# TeaGram Shop Web App

Интерфейс web app интернет-магазина китайского чая в стиле Telegram. Реализованы ключевые экраны MVP с поддержкой светлой и тёмной темы, локального хранения корзины и темы.

## Функционал

- Каталог с фильтром по категориям и карточками товаров.
- Страница товара с выбором фасовок и управлением количеством.
- Корзина с редактированием позиций и расчётом итогов.
- Оформление заказа с валидацией и имитацией отправки на бэкенд.
- Экран подтверждения заказа с кратким резюме.
- Переключение темы и сохранение состояния в `localStorage`.

## Запуск проекта

```bash
npm install
npm run dev
```

Сборка для продакшена:

```bash
npm run build
```

> ⚠️ Для запуска в песочнице может потребоваться настроить прокси для доступа к npm.
